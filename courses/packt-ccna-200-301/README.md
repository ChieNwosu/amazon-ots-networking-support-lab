## What This Course Covers

This specialization aligns with the Cisco CCNA 200-301 exam objectives, covering the full scope of associate-level networking knowledge. Topics include network access, IP connectivity, IP services, security fundamentals, automation, and programmability. The program is structured to develop comprehensive networking competency at the professional certification level.

## OTS Role Connection

The CCNA curriculum directly maps to the technical depth expected of IT support engineers working with enterprise network infrastructure. Amazon OTS engineers troubleshoot connectivity issues, support network device operations, and collaborate with network teams on infrastructure maintenance. CCNA-level knowledge provides the diagnostic depth needed to perform root-cause analysis on network issues affecting FC operations and to communicate technical findings clearly during incident response.

## Key Topics

- Network access (VLANs, EtherChannel, spanning tree)
- IP connectivity (routing protocols, static and dynamic routing)
- IP services (DHCP, DNS, NAT, NTP, SNMP)
- Security fundamentals (ACLs, port security, AAA)
- Automation and programmability (REST APIs, configuration management)
- Network fundamentals (TCP/IP, OSI, cabling, topologies)
- Wireless networking (standards, configuration, security)
