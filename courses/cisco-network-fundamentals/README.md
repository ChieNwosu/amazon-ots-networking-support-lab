## What This Course Covers

This specialization provides foundational knowledge in computer networking concepts directly from Cisco. It covers the OSI and TCP/IP models, IP addressing and subnetting, Ethernet and switching fundamentals, routing concepts, and network device configuration. The program is designed to build the core understanding needed for network support and administration roles.

## OTS Role Connection

Amazon IT Support Engineer I (OTS) responsibilities include troubleshooting network connectivity issues in fulfillment center environments, supporting network infrastructure that enables operations, and diagnosing Layer 1-3 issues that affect production uptime. This course builds the foundational knowledge required to identify, isolate, and resolve common network problems in a structured environment.
