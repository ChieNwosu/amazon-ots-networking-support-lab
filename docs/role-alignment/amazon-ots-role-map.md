# Amazon OTS Role Map

## Overview

This document maps the responsibilities of an Amazon IT Support Engineer I, Ops Tech Solutions (OTS) role to the learning artifacts and coursework documented in this repository. The purpose is to show clear alignment between structured technical preparation and the skills required to support fulfillment center IT infrastructure.

## Amazon FC Operations Experience Connection

Working in an Amazon Fulfillment Center provides foundational awareness that supports OTS technical responsibilities:

| FC Operations Awareness | OTS Technical Application |
|---|---|
| Understanding of operational uptime requirements | Prioritizing issues based on production impact |
| Familiarity with FC floor layout and workflow | Faster physical troubleshooting and device location |
| Experience with shift-based operations | Readiness for 24/7 IT support coverage models |
| Exposure to scanning, conveyor, and sortation systems | Contextual awareness of devices OTS supports |
| Understanding of peak volume and critical periods | SLA-aware incident prioritization |
| Process discipline and SOP adherence | Structured approach to IT support procedures |

**Important note:** FC operations experience provides contextual awareness and transferable soft skills. This repository documents technical skill development through structured coursework, not production IT support experience.
